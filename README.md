I am an **AI Engineer & Generative AI Developer** focused on building intelligent applications, natural language processing tools, and robust backend services. I specialize in designing **Retrieval-Augmented Generation (RAG) systems**, developing interactive AI agents, and engineering backends with **FastAPI**.

## 🔥 Featured Projects

### 📄 Document Intelligence Platform
* **Description:** A full-stack application that enables users to upload PDF documents and interact with them using a RAG (Retrieval-Augmented Generation) pipeline.
* **Features:** Automatic document summarization, multi-choice quiz generation, and a context-grounded conversational chat interface.
* **Tech Stack:** React (Vite), FastAPI, FAISS (local vector storage), HuggingFace Embeddings, Groq API (LLaMA/Mixtral), PostgreSQL, SQLAlchemy.
* [🔗 Repository](https://github.com/saiteja-025/docai-project)

---

### 👁️ VisionGuard AI
* **Description:** A web application for automated Image Quality Assessment (IQA) and blur detection using computer vision algorithms and machine learning classification.
* **Features:** Extracts blur, spectral motion blur, brightness tails, contrast, and high-frequency noise metrics. Programmatically generates PDF reports and CSV exports.
* **Tech Stack:** Python, Flask, OpenCV (Headless), NumPy, Scikit-Learn (Random Forest), SQLite, ReportLab, Docker.
* [🔗 Repository](https://github.com/saiteja-025/VisionGuard-AI)

---

### 💰 AI Expense Analytics Agent
* **Description:** A Telegram bot designed to help users log and track daily expenses using natural language text parsing.
* **Features:** Text message intent extraction, category-based categorization, and summary generation.
* **Tech Stack:** Python, python-telegram-bot, Telegram Bot API.
* [🔗 Repository](https://github.com/saiteja-025/AI-expense-agent-telegram-bot)

---

### 🎤 NLP-Based AI Interview Simulator
